# Day Planner
## About
A simple weather dashboard to check a weekly forecast of places from around the world
## How to use
1.  Enter your desired location into the search field.
2.  Either use the enter key or the search button to initiate the search.
3.  After searching, your specified locations' weather data will appear beneath the search field.
4. The last 5 searched places are created as buttons for you under the search field for easier access.
